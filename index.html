The Key Insight

HFM, Planning, and Tax Provision are NOT separate products.

They are pre-built applications that Oracle constructed on top of Essbase. The Java containers provide the UI, workflow, and business logic, but all data lives in Essbase cubes and all calculations run in the C++ Essbase engine.
